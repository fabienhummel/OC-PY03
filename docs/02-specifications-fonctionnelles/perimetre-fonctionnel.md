# Périmètre fonctionnel de HomeSkolar

## Périmètre inclus

### 1. Gestion du compte utilisateur

Pour les élèves et les tuteurs bénévoles :

- s'inscrire ;
- se connecter ;
- gérer leur mot de passe ;
- gérer leurs données personnelles.

### 2. Communication entre l'élève et son tuteur

- échanger et consulter des messages ;
- permettre à l'élève d'épingler certains messages ;
- notifier l'utilisateur lorsqu'un message n'a pas été lu.

### 3. Rencontres et calendrier

- planifier les rencontres ;
- consulter les rendez-vous et événements ;
- afficher ces éléments dans une page de calendrier classique.

### 4. Gestion des tâches

- consulter les tâches à réaliser avant la prochaine rencontre ;
- être notifié de ces tâches dans l'application ;
- créer une tâche personnelle ;
- utiliser une tâche personnelle comme note ou mémo.

## Acteurs inclus

- élève ;
- tuteur bénévole.

## Éléments exclus

Les éléments suivants sont exclus des spécifications fonctionnelles, car ils ne sont pas explicitement demandés :

- rôle administrateur ;
- règles détaillées d'affectation entre élève et tuteur ;
- messagerie de groupe et pièces jointes ;
- visioconférence ;
- règles détaillées de modification ou d'annulation d'un rendez-vous ;
- rappels de rendez-vous et gestion des fuseaux horaires ;
- priorité, échéance ou statut détaillé des tâches ;
- choix d'architecture, d'hébergement ou de déploiement.

## Règles de rédaction

- décrire ce que l'utilisateur pourra faire ;
- employer un langage simple et non technique ;
- découper les fonctionnalités en sous-fonctionnalités ;
- éviter les répétitions ;
- réserver les fonctionnalités supplémentaires à d'éventuelles évolutions futures.
