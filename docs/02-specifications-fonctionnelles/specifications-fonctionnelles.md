# Spécifications fonctionnelles

Cette section présente les tâches que l'élève et le tuteur bénévole pourront réaliser dans HomeSkolar. Elle reprend uniquement les besoins explicitement formulés dans la note de lancement.

## 1. Gestion du compte utilisateur

L'élève et le tuteur bénévole pourront :

- s'inscrire ;
- se connecter ;
- gérer leur mot de passe ;
- gérer leurs données personnelles.

## 2. Communication

L'application proposera un système de communication simple entre l'élève et son tuteur.

Les utilisateurs pourront :

- échanger et consulter des messages ;
- identifier les messages non lus grâce à une notification.

L'élève pourra également épingler certains messages afin de les retrouver facilement.

## 3. Rencontres et calendrier

L'élève et le tuteur pourront planifier leurs rencontres et consulter leurs rendez-vous.

La page de calendrier permettra :

- d'afficher les rendez-vous ;
- d'afficher les autres événements de l'utilisateur ;
- de consulter l'ensemble dans une vue de calendrier classique.

Les rencontres sont prévues régulièrement, sous la forme de courts rendez-vous hebdomadaires.

## 4. Gestion des tâches

À l'issue d'une rencontre, l'élève pourra consulter la liste des tâches à réaliser avant la rencontre suivante. Ces tâches seront signalées dans l'application afin de faciliter son suivi.

L'élève et le tuteur pourront également :

- créer une tâche pour eux-mêmes ;
- utiliser une tâche personnelle comme note ou mémo.

## Limites du périmètre

Aucune fonctionnalité non explicitement mentionnée dans la note de lancement n'est intégrée à cette rédaction. Les choix techniques, règles détaillées et évolutions éventuelles seront traités dans les étapes concernées du projet.
